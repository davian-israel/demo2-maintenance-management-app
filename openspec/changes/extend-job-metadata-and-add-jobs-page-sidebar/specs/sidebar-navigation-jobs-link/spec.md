## ADDED Requirements

### Requirement: Application SHALL include sidebar navigation
The UI SHALL provide a persistent sidebar menu for primary navigation destinations.

#### Scenario: Render sidebar
- **WHEN** a user opens any main application page
- **THEN** a sidebar menu is visible with primary navigation links

### Requirement: Sidebar SHALL include Jobs link
The sidebar SHALL include a menu item that routes to the dedicated jobs page.

#### Scenario: Navigate to jobs from sidebar
- **WHEN** a user clicks the Jobs sidebar link
- **THEN** the app navigates to the jobs page and displays the jobs list
